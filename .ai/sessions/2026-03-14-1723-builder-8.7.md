## What Was Done

- Switched the non-JSON, non-`--docs` `launchpad config show` path to print a
  Rich TOML syntax renderable instead of plain `click.echo(...)` text.

  `launchpad config show --docs` as the plain schema-doc output.
- Added regression coverage for the highlighted human path, the no-color
  console wiring, and the unchanged JSON/docs branches, then updated the
  config command/reference docs accordingly.
