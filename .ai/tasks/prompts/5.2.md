# Task Prompt: 5.2 - Hybrid transfer engine and CLI surfaces

## Objective

Implement the chosen Phase 5 hybrid transfer system so Launchpad can move data
using both striped single-payload transfer and worker-pool many-file transfer,
with a coherent CLI surface across submit and download.

## Context

- Task `5.1` now leaves the implementation-ready architecture decision for
  `single-file`, `multi-file`, and `auto` transfer modes.
- The current transfer engine lives in `src/launchpad_cli/core/transfer.py`,
  with configuration in `src/launchpad_cli/core/config.py` and integration
  points in `src/launchpad_cli/cli/submit.py` and
  `src/launchpad_cli/cli/download.py`.
- The roadmap still prioritizes correctness, resume support, and integrity over
  raw throughput. The implementation should take the aggressive path selected
  by the human without leaving open decisions about safety, fallback, or
  operator control.

## In Scope

- Add the new transfer-mode surface to `launchpad submit` and
  `launchpad download`, including `auto`, `single-file`, and `multi-file`
  selection plus `--streams`.
- Implement striped single-file transfer using multiple SSH/SFTP workers with
  safe assembly/resume behavior for submit and download.
- Implement multi-file transfer using a worker pool of whole-file transfers,
  preserving directory structure and per-file resume/checksum behavior.
- Apply the agreed auto-selection rules, explicit override behavior, and
  effective-stream fallback when the server will not allow the requested worker
  count.
- Add or update tests and documentation that cover the new transfer surface,
  resume guarantees, fallback behavior, and user-facing summaries/errors.

## Out of Scope

- CLI typo suggestions or broad `--json` coverage work assigned to later tasks.
- Logging audit beyond the modules directly touched by the transfer changes.
- Solver-specific feature work.
- Leaving unresolved architecture or CLI-surface questions for later tasks.

## Deliverables

- `src/launchpad_cli/core/transfer.py`
- `src/launchpad_cli/core/config.py`
- `src/launchpad_cli/cli/submit.py`
- `src/launchpad_cli/cli/download.py`
- Any supporting transport helpers or display updates needed for the new mode
  and stream summaries
- Associated tests and documentation updates
