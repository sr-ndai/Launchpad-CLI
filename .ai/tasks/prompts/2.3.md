# Task Prompt: 2.3 - Submit orchestration and dry-run UX

## Objective

Wire the first functional `launchpad submit` command around the established
solver, transfer, remote, and SLURM primitives, including a useful dry-run
preview.

- Phase 2’s milestone is a working Nastran submission path from the local CLI.
- `.ai/plan.md` sections 3.1, 4.2, 4.3, 5.1, 7.1, 8, 12, and 15 describe the
